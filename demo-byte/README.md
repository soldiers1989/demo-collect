# 字节相关记录

## 规则

有符号

1、二进制的最高位为符号未，0表示整数，1表示负数    
2、正数的原码、反码、补码都一样    
3、负数的反码 = 符号位不变，其它位取反    
4、负数的补码 = 负数反码 + 1    
5、0 的反码、补码都为0；    
6、java没有无符号数，java中都是有符号的    
7、在计算机运算的啥时候，都是以补码的方式来运算的；    

基本概念

java中有4个位运算 & | ^ -

1、按位与& ： 两位为1，结果为1    
2、按位或| ：两位又一个为1，结果为1；   
3、按位异或 ：两位一个为0一个为1，结果为1;

问：运算主要用户啥场景了？

移动运算： >> <<     

1、算数右移：低位溢出，符号位不变，并用符号位补溢出高位   
2、算术左移：符号位不变，低位补0


无符号右移 `>>>`
